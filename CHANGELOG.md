# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-01

### Added
- `Picture` component for Vue 3
- Vite plugin for automatic image compression at build time
- Configurable compression percentage support
- Automatic WebP version generation for images
- Support for JPEG, PNG, WebP and SVG formats
- TypeScript types
